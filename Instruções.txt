instrução para vizualizar o tema do site

requisitos:
-node instalar https://nodejs.org/dist/v18.12.1/node-v18.12.1-x64.msi

para abrir ele voce deve fazer o seguinte.
-instalar o node js.
-criar uma pasta dentro do disco local c chamada Dev.
-extrair o arquivo dentro da pasta Dev.
abriri o cmd ou prompt de comando como adiministrador.
em seu prompt escreva: 
cd..
cd..
cd Dev
cd Desapego
npm install
npm start

nisto ira abrir uma pagina no navegador.
